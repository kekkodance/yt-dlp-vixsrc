# yt-dlp-vixsrc

A yt-dlp extractor plugin for vixsrc.to.

## Features
- Fast chunk downloading for VixSrc HLS streams
- TMDB metadata for clean movie/episode titles

## Installation

python -m pip install -U https://github.com/kekkodance/yt-dlp-vixsrc/archive/master.zip

Or install locally:
python -m pip install .
